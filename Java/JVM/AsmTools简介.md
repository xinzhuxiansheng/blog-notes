## 安装
1. jdk1.8 及环境变量配置
2. ant 1.10.x 必须要求jdk1.8及以上，及环境变量配置
3. mercurial安装

```shell
#利用mercurial克隆代码
hg clone http://hg.openjdk.java.net/code-tools/asmtools

#在asmtools/build目录下，执行ant命令
cd asmtools/build
ant
```
